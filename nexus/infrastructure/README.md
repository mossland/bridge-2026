# Infrastructure

Infrastructure는 시스템 인프라 컴포넌트를 포함합니다.

## 개요

이 디렉토리는 BRIDGE 2026 시스템의 인프라 레벨 컴포넌트를 포함합니다.

## 구조

- `event-bus/`: 레이어 간 비동기 이벤트 통신
- `database/`: 데이터베이스 스키마, 마이그레이션, 시드 데이터
- `monitoring/`: Prometheus 및 Grafana 설정

## 역할

- **이벤트 버스**: 레이어 간 느슨한 결합을 위한 이벤트 기반 통신
- **데이터베이스**: 전체 시스템의 데이터 영속성
- **모니터링**: 시스템 가시성 및 운영 메트릭
