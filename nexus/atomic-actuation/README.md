# Atomic Actuation

Atomic Actuation은 거버넌스 통과 시 온체인/오프체인 실행을 원자적으로 트리거하는 레이어입니다.

## 개요

Human Governance에서 제안이 통과되면, Atomic Actuation이 실행을 담당합니다. 모든 실행은 원자적(atomic)으로 처리되어 부분 실행을 방지합니다.

## 구현 범위

### MVP v0

#### 온체인 실행

- **파라미터 변경**: 거버넌스 파라미터 업데이트
- **트레저리 집행**: 멀티시그/타임락을 통한 안전한 자금 이동
- **BridgeLog 앵커링**: 실행 결과를 BridgeLog 컨트랙트에 기록

#### 오프체인 실행

- **GitHub 이슈 생성**: 개발 작업 자동화
- **캠페인 배포**: 마케팅 캠페인 자동 실행
- **공지 자동 작성**: 커뮤니티 공지사항 자동화

### 확장 계획

#### 2027+: Digital Twin 확장

- BMS/에너지 최적화 API 어댑터
- 빌딩/시설 센서 제어

#### 2028+: Humanoid/로봇 확장

- 로봇 태스크 할당
- 안전정책 적용

## 실행 프로세스

1. **제안 통과 확인**: Human Governance에서 투표 통과 확인
2. **실행 계획 검증**: Decision Packet의 실행 계획 검증
3. **원자적 실행**: 모든 실행을 원자적으로 처리
4. **결과 기록**: 실행 결과를 BridgeLog에 앵커링
5. **Proof of Outcome 전달**: 실행 결과를 Proof of Outcome 레이어로 전달

## 안전장치

- **롤백 계획**: Decision Packet에 포함된 롤백 계획 실행 가능
- **멀티시그**: 트레저리 집행은 멀티시그 필요
- **타임락**: 시간 기반 제한
- **실행 검증**: 실행 전 모든 조건 재검증

## 기술 스택

- **온체인**: Solidity, ethers.js
- **오프체인**: Node.js/TypeScript, GitHub API, 외부 서비스 API
- **원자성 보장**: 트랜잭션 관리, 보상 트랜잭션
